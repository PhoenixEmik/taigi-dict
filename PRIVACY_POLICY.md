本文件為 `台語辭典` App（`taigi_dict`、Android application ID `org.taigidict.app`、iOS bundle identifier `org.taigidict.app`）與本專案 repo 使用的雙語隱私權政策。

## English

### Summary

`台語辭典` is an offline Taiwanese Hokkien and Mandarin dictionary app. The app does not require account registration and is not designed to sell personal data or track users across third-party apps or websites.

The app stores core dictionary resources and user preferences on the user's device so that dictionary lookup and audio playback can work offline after setup.

### Information Stored On Device

The app may store the following information locally on your device:

- app preferences such as language, theme, and reading text size
- bookmark data
- recent search history
- downloaded offline resources, including dictionary source files, generated local database files, and offline audio archives

This local data is used to provide app functionality. It is not uploaded by the app to the developer merely because it exists on your device.

### Network Use

The app uses network access for limited product functions, including:

- downloading offline dictionary source files
- downloading offline audio archives
- opening external links selected by the user, such as the project repository or Ministry of Education reference pages
- sharing content only when the user explicitly invokes the system share action

The current production download endpoints used by the app are hosted under `https://app.taigidict.org/assets/`.

### Information That May Be Processed During Network Requests

When the app downloads files or opens external links, the relevant hosting provider, CDN, or destination website may receive technical request data as part of normal internet communication. This can include:

- IP address
- request timestamps
- requested URL
- basic network and protocol metadata

This project does not use that technical traffic to create in-app user accounts, profile users inside the app, or provide cross-app advertising profiles.

### Permissions

On Android, the release build currently uses these normal permissions:

- `INTERNET`
- `ACCESS_NETWORK_STATE`

These permissions are used to support file downloads, link handling, and network-aware media behavior.

The app does not currently request high-risk Android permissions such as location, camera, microphone, contacts, SMS, call logs, or photo/media library access.

### Data Sharing

The app does not sell personal information.

The app does not share personal information with third parties for advertising or cross-app tracking.

User-initiated sharing is different: if you choose to share a dictionary entry through the system share sheet, the selected receiving app or service will process the shared content under its own privacy terms.

### Third-Party Services And Open Source Components

The app uses third-party and open source components for core functionality, including network transfer, audio playback, native sharing, local storage, and external link launching.

The app may also connect to third-party websites that the user chooses to open, including:

- `taigidict.org`
- `app.taigidict.org`
- `github.com`
- `sutian.moe.edu.tw`

Each external site operates under its own terms and privacy practices.

### Data Retention And Deletion

Most app data is stored locally on the device.

You can generally remove locally stored app data by:

- uninstalling the app
- clearing the app's local storage through the operating system
- using app settings to manage certain downloaded offline resources

If the developer receives technical server logs through normal hosting operations, those logs are retained according to the hosting provider's operational policies.

### Children's Privacy

The app is a general reference tool and is not designed to knowingly collect personal information from children.

### Changes To This Policy

This Privacy Policy may be updated when the app's behavior, infrastructure, or compliance requirements change. The latest version in this repository supersedes prior repository copies.

### Contact

Issue tracker:

- `https://github.com/PhoenixEmik/taigi-dict/issues`

## 正體中文

### 摘要

`台語辭典` 是一個以離線使用為核心的台語 / 華語辭典 App。App 不需要註冊帳號，也不是以出售個人資料或跨第三方 App / 網站追蹤使用者為目的而設計。

App 會將核心詞典資源與使用者偏好儲存在使用者裝置上，讓辭典查詢與音檔播放能在初始化後離線使用。

### 儲存在裝置上的資料

App 可能會在你的裝置本機儲存以下資料：

- 語言、主題、字級等 App 偏好設定
- 書籤資料
- 最近搜尋紀錄
- 已下載的離線資源，包括詞典原始檔、產生出的本機資料庫檔案，以及離線音檔壓縮檔

這些本機資料用於提供 App 功能。它們存在於你的裝置上，並不代表 App 會自動將其上傳給開發者。

### 網路使用

App 僅在有限的產品功能中使用網路，包括：

- 下載離線詞典原始檔
- 下載離線音檔壓縮資源
- 開啟使用者主動點選的外部連結，例如專案 repo 或教育部參考頁面
- 僅在使用者明確觸發系統分享動作時分享內容

App 目前使用的正式環境下載端點位於 `https://app.taigidict.org/assets/`。

### 網路請求期間可能處理的資料

當 App 下載檔案或開啟外部連結時，相關的主機服務商、CDN 或目標網站，可能會在正常網路通訊過程中接收技術性請求資料，例如：

- IP 位址
- 請求時間
- 請求的 URL
- 基本網路與通訊協定中介資料

本專案不會以這些技術性流量在 App 內建立使用者帳號、建立使用者輪廓，或進行跨 App 廣告追蹤。

### 權限

Android release 版本目前使用以下一般權限：

- `INTERNET`
- `ACCESS_NETWORK_STATE`

這些權限用於支援檔案下載、外部連結處理，以及依網路狀態調整的媒體行為。

App 目前不會要求位置、相機、麥克風、聯絡人、簡訊、通話紀錄、相簿 / 媒體庫等高風險 Android 權限。

### 資料分享

App 不會販售個人資料。

App 不會為了廣告或跨 App 追蹤而向第三方分享個人資料。

但使用者主動分享是另一件事：如果你透過系統分享面板主動分享詞條內容，接收該內容的 App 或服務會依其自身隱私條款處理資料。

### 第三方服務與開源元件

App 會使用第三方與開源元件提供核心功能，包括網路傳輸、音訊播放、原生分享、本機儲存與外部連結開啟。

使用者也可能主動連線至外部網站，包括：

- `taigidict.org`
- `app.taigidict.org`
- `github.com`
- `sutian.moe.edu.tw`

各外部網站均有自己的服務條款與隱私實務。

### 資料保留與刪除

多數 App 資料都儲存在使用者裝置本機。

你通常可以透過以下方式移除本機資料：

- 解除安裝 App
- 在作業系統中清除 App 本機儲存空間
- 在 App 設定中管理部分已下載的離線資源

若開發者因一般主機營運而接收到技術性伺服器日誌，該類日誌會依主機服務商的營運政策保留。

### 兒少隱私

本 App 是一般參考工具，並非以主動蒐集兒少個人資料為目的而設計。

### 政策變更

當 App 行為、基礎設施或合規需求調整時，本隱私權政策可能更新。repo 中的最新版本優先於舊版文件。
